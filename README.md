# ⛅ SkyPulse — Real-Time Weather Dashboard

A sleek, real-time weather intelligence app that lets you search any city worldwide and instantly view detailed weather conditions, hourly forecasts, and a 5-day outlook.

## ✨ Features

- 🔍 **City Search** — Search any city worldwide with instant results
- 📍 **Geolocation** — Auto-detect your current location with one click
- 🌡️ **Current Conditions** — Temperature, feels like, high/low, description
- 💧 **8 Weather Metrics** — Humidity, wind speed, visibility, pressure, cloud cover, rain, sunrise & sunset
- ⏱️ **24-Hour Hourly Forecast** — Hour-by-hour breakdown with rain probability
- 📅 **5-Day Forecast** — Daily high/low, conditions, and precipitation chance
- ☀️ **UV Index Bar** — Visual UV scale with severity level
- 🌙 **Day/Night Icons** — Dynamic weather emojis based on time of day
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop

## 🚀 Getting Started

### Option 1 — Open Directly
Just open `index.html` in any browser. No build tools or dependencies needed.

### Option 2 — Clone the Repo
```bash
git clone https://github.com/jayu2236j/SkyPulse-Weather-App.git
cd SkyPulse-Weather-App
open index.html
```

### API Key
This project uses the [OpenWeatherMap API](https://openweathermap.org/api). The key is already included for demo purposes. To use your own:
1. Sign up free at [openweathermap.org](https://openweathermap.org/register)
2. Get your API key from the dashboard
3. Replace the `KEY` variable in `index.html`
